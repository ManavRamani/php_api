1. **Users API:**
   - Fetch all users: `GET` request to `http://localhost/exam_php_api/fetch_all_users.php`
   - Fetch user by ID: `GET` request to `http://localhost/exam_php_api/fetch_user_by_id.php?id={user_id}`
   - Insert user: `POST` request to `http://localhost/exam_php_api/insert_user.php` with body parameters `username`, `email`, `password`
   //----- Update user: `PUT` request to `http://localhost/exam_php_api/update_user.php?id={user_id}` with body parameters `username`, `password`
   - Delete user: `DELETE` request to `http://localhost/exam_php_api/delete_user.php?id={user_id}`
